## 기본 개념

### 사용하기 쉽다는 특징을 기반으로 가져감.
CSS는 원하는 스타일마다 따로따로 이름을 붙여서 각각의 요소에 적용함.  

TailwindCSS는 원하는 속성이 구현된 클래스를 HTML에 적용할 수 있음.  
처음에 보면 지저분할 수 있지만 여러 장점이 있다.
- 클래스 이름을 짓는지 여부를 선택할 수 있다.
- CSS 파일의 덩치가 커지지 않는다.
- 내가 바꿔도 다른 곳의 영향을 안 준다.

설정파일에서 상세한 설정을 할 수 있다. 
[설정 설명](https://tailwindcss.com/docs/configuration)
